HolaCiné – Site de recommandation de films

Développement full-stack d'une plateforme de recommandation de films, de l'analyse à la conception, en passant par l'implémentation d'un algorithme de similarité cosinus. Technos : HTML/CSS/JS, Node.js, Python, MySQL.

Dossier d'analyse

Ce dossier répond à un appel d'offres fictif pour le développement d'un site de recommandation de films. Il détaille la méthodologie, les choix technologiques, et les justifications pour le client, en utilisant un langage clair et accessible.

Objectifs :

  • Définir les besoins fonctionnels et non-fonctionnels.
  • Proposer une architecture technique adaptée.
  • Planifier les étapes de développement.

Dossier de conception

Le dossier de conception est destiné aux développeurs et contient :

  • Des diagrammes UML (cas d'utilisation, classes, séquence).
  • Une architecture technique détaillée (front-end, back-end, base de données).
  • Les algorithmes utilisés (recommandation par similarité cosinus).
  • Les APIs et flux de données.

Ce dossier a permis de structurer le développement et d’assurer une cohérence entre les membres de l’équipe.

Stack technique

Pour ce projet, j’ai utilisé une combinaison de technologies pour le front-end et le back-end :

  • Front-end : HTML5, CSS3, JavaScript (vanilla) pour une interface réactive et moderne.
  • Back-end : Node.js pour la logique serveur et Python pour l’algorithme de recommandation (similarité cosinus).
  • Base de données : MySQL pour stocker les films, utilisateurs et préférences.
  • Outils : MAMP pour l’environnement local, Git pour le versioning.

Ce projet m’a permis de renforcer mes compétences en :

  • Développement full-stack.
  • Conception d’algorithmes de recommandation.
  • Travail en équipe et gestion de projet.
